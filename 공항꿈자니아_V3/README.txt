공항 꿈자니아 V3

V3 주요 기능
- 태블릿 한 화면 중심의 컴팩트 레이아웃
- 공항 출발 전광판 시작 화면
- 직업 선택 4개 2×2 배치
- 다음 차례 친구가 직접 누르는 입장하기 버튼
- 부드러운 공항 안내음과 한국어 음성 안내
- 열매1반 15명 이름 버튼
- 이름 버튼을 한 번 누르면 즉시 대기번호 발급
- 직업별 독립 대기번호 및 관리자 관제실
- PWA 설치 및 오프라인 캐시

Vercel 배포
1. ZIP 압축을 풉니다.
2. index.html, style.css, app.js 등이 바로 보이는 폴더를 업로드합니다.
3. 새 프로젝트로 올리거나 Git 저장소에 연결하여 배포합니다.
